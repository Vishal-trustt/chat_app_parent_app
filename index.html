Integration Guide

Learn how to integrate the Trustt chatbot into your own website with these simple steps.

Step 1: Add the Container

First, add the chatbot container to your HTML:

<div id="trustt-chatbot"></div>

Step 2: Include the Script

Add the Trustt chatbot script to your HTML head or before the closing body tag:

<script data-chatbot-script src="https://dev-tgpt.trustt.com/bot/static/js/main.js"></script>

Step 3: Configuration (Optional)

You can customize the chatbot appearance and behavior:

<script> // Initialize chatbot with custom settings window.trusttChatbot = { theme: 'light', position: 'bottom-right', language: 'en' }; </script>

CSS Styling

Style the chatbot container as needed:

#trustt-chatbot { position: fixed; bottom: 20px; right: 20px; z-index: 1000; }
